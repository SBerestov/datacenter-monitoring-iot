Группа: __РИС-23-1бзу.__

Выполнил: __Берестов Савелий__

## Запланированный функционал для 1 семестра предмета:
1. Главная страница мониторинга с дашбордом. 
2. Кнопка обновления актуальных данных и табличного вида. 
3. Отображение данных с датчиков в реальном времени. 
4. Список доступных датчиков.
5. Отрисовка временного графика по выбранному параметру датчика.
6. Предупреждение/подсвечивание выхода за предела параметра.
