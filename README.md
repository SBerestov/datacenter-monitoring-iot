# Система мониторинга ЦОД на основе интернета вещей
Группа: __РИС-23-1бзу.__

Выполнил: __Берестов Савелий__

## Запланированный функционал для 1 семестра предмета:
1. Главная страница мониторинга с дашбордом. 
2. Кнопка обновления актуальных данных и табличного вида. 
3. Отображение данных с датчиков в реальном времени. 
4. Список доступных датчиков.
5. Отрисовка временного графика по выбранному параметру датчика.
6. Предупреждение/подсвечивание выхода за предела параметра.

## Запланированный функционал для 2 семестра предмета:
1. Выбор временного диапазона отображения данных
2. 3 инпут поля времени по дням, неделям и месяцам (по дефолту за все время)
3. Вторая страница "Дашборд"
4. Список всех объектов на странице "Дашборд" с информацией о текущем статусе и возможностью нажать на него для перехода в "Детальный вид"
5. Подключение API-карты от Яндекс, на карте видны все местоположения объектов и статус
